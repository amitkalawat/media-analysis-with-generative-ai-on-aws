**Department:** Digital Operations
**Last Updated:** March 2025

## Video Synopsis Requirements

### 1. Title Standards
- Films: Title include name of the film
- Sports: Title include name of the sports venue
- Use title case

### 2. Description Format
- Description is less than 200 words
- Must include cast and their roles (Films) or teams and athletes (Sports)
- Structure: Intro, body, call-to-action

### 3. OUTPUT SCHEMA
```json
{
    "title": "[Catchy title for video]",
    "description": "[Insert Video Description]"
}
```
